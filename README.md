# Product Inventory Mini CRUD API

A lightweight **RESTful API** for managing products in an inventory system, built with **Node.js, TypeScript, Express, and MongoDB (Mongoose)**, following **OOP principles** (Repository → Service → Controller).  

This API allows you to **create, read, update, delete products**, and **manage stock quantities**.

---

## Features

- Create new products  
- Retrieve single or multiple products  
- Update product details  
- Delete products  
- Increase or decrease product stock  
- Automatic timestamps (`createdAt` and `updatedAt`)  
- Scalable OOP folder structure  

---

## Tech Stack

- **Backend:** Node.js, Express  
- **Language:** TypeScript  
- **Database:** MongoDB with Mongoose  
- **Architecture:** Repository → Service → Controller (OOP)
